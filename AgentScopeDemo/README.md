File Descriptions

- main_cn.py - Main game logic and controllers

- prompt_cn.py - Chinese prompt word management

- game_roles.py - Game character definition and configuration

- structured_output_cn.py - Structured output model definition

- utils_cn.py - Game utility functions and helper methods

- README.md - This documentation

Project Features

- Message-Driven Architecture: Demonstrates AgentScope's core messaging mechanism
- Concurrent Collaboration: Showcases real-time interaction between multiple agents online simultaneously
- Role-Playing: Each agent possesses a dual identity (game character + Three Kingdoms figure)
- Structured Output: Constrains agent behavior through a Pydantic model
- Fault Tolerance Mechanism: An exception in a single agent does not affect the overall game flow
